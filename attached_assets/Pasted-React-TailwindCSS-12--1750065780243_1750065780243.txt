React + TailwindCSS 기반의 웹사이트를 만들어줘.  
홈페이지에서는 총 12개의 심리 테스트를 카드 형식으로 보여주고, 클릭 시 각 테스트의 별도 경로로 이동할 수 있어야 해.  
각 테스트는 시작 페이지 → 질문 페이지(10문항) → 결과 페이지로 구성되고, 결과는 4가지 유형 중 하나로 분류돼.

📌 요구사항 요약:
- 프론트엔드: React + TailwindCSS
- 라우팅: React Router 사용
- 반응형 디자인 필수 (모바일 최적화)
- 메인 페이지는 테스트 목록을 카드 형태로 구성
- 각 테스트는 독립적으로 작동 (질문은 JSON, 결과는 점수 기반 분기)
- 결과 페이지에는 유형 이름, 설명, 간단한 조언 포함
- 다크모드 필수 사항
- 공유 버튼 필수 사항
- 서버없이 프론트엔드에서만 동작하도록 Static 하게 Claoudflare 에 Page 만 업로드할예정


---

🎯 테스트 목록 (총 12개):

1. **AI 적응 유형 테스트** 🤖  
   경로: `/test/ai-adapt`  
   주제: AI와 협업하는 능력, AI시대의 적응력과 직업 성향  

2. **연애 회피/집착 성향 테스트** ❤️‍🔥  
   경로: `/test/romance-attachment`  
   주제: 애착 유형과 연애 시 행동패턴(감정 표현, 밀당, 회복력 등)  

3. **디지털 감정지능 테스트 (Digital EQ)** 💬  
   경로: `/test/digital-eq`  
   주제: 메신저, 댓글 등 온라인 환경에서의 감정 인식 능력 측정  

4. **내 인생 시나리오 유형 테스트** 🎬  
   경로: `/test/life-scenario`  
   주제: 인생을 영화 장르/구성으로 비유해 성향 분석 (예: 성장형, 로맨스형 등)  

5. **사회적 에너지 유형 테스트** 🌱  
   경로: `/test/social-energy`  
   주제: 어디서 에너지를 얻고, 소모하는지 (확장된 I/E 모델)  

6. **성공의 장애물 유형 테스트** 🧱  
   경로: `/test/success-blocker`  
   주제: 게으름, 미루기, 두려움 등 성공을 막는 가장 큰 내부 요인 분석  

7. **나의 생산성 루틴 테스트** ⏰  
   경로: `/test/productivity-routine`  
   주제: 집중 유형, 시간대, 도구 등 나만의 생산성 루틴 분석  

8. **가상현실 적응력 테스트** 🥽  
   경로: `/test/vr-adapt`  
   주제: 메타버스나 가상 공간에 대한 몰입/현실 혼동 경향  

9. **인터넷 밈 유형 테스트** 😂  
   경로: `/test/meme-type`  
   주제: 밈을 통해 세대 코드, 유머 감성, 디지털 감각 분석  

10. **디지털 자아 분리 테스트** 🌐  
    경로: `/test/digital-self`  
    주제: 현실 자아 vs 온라인 자아 간 괴리 측정  

11. **마음속 목소리 유형 테스트** 🧠  
    경로: `/test/inner-voice`  
    주제: 내면의 심리적 목소리(비판자, 격려자, 회피자 등) 중 지배적인 성향  

12. **스트레스 해소 방식 테스트** 😮‍💨  
    경로: `/test/stress-type`  
    주제: 스트레스 상황에서의 반응 패턴 (말하기/회피/도피/자기복구 등)

---

🧩 각 테스트 결과 유형은 4가지이며, JSON 질문 데이터로 구성되며 점수 합산 방식으로 결과를 분기해.  
결과 페이지에는 유형 이름, 요약 설명, 공감 문구(또는 밈 이미지 placeholder)를 함께 표시해줘.

🎨 디자인 키워드:
- 메인 페이지: 그리드 카드형 테스트 선택 UI (12개)
- 색상 테마: 라이트 톤, 심플 & 따뜻한 UX (light gray, skyblue, beige 등)
- TailwindCSS로 카드 간격, 그림자 효과, 아이콘 배치 정돈
- 테스트별 캐릭터 이모지나 아이콘 함께 사용

🛠️ 기술 구성:
- React
- React Router
- TailwindCSS
- 각 테스트에 대한 JSON 기반 질문/결과 시스템
- 결과별 공유 기능 (선택)
- 모바일 퍼스트 디자인
